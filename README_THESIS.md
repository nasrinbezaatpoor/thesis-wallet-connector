# 🎓 پایان‌نامه: Web3Connector — پل ارتباطی به دنیای Web3
### بدون API Key، فقط با آدرس بلاکچین

---

## ❓ مشکل: می‌خوای به Web3 وصل بشی ولی API Key نداری

تصور کن:
- یه آدرس ولت داری، می‌خوای ببینی توی بلاکچین چقدر پول داری
- دو تا آدرس داری، می‌خوای ببینی به هم ربط دارند یا نه
- می‌خوای ببینی یه آدرس قرارداد هوشمنده یا ولت معمولی
- و مهم‌تر از همه **بدون اینکه مجبور باشی برای API Key ثبت‌نام کنی**

**اینجاست که این پروژه به کارت میاد.** فقط با یه آدرس (مثل `0x123456...`) کار می‌کنه و مستقیم از بلاکچین اطلاعات می‌گیره.

---

## ✨ چیزایی که می‌تونی باهاش انجام بدی

### 🏦 `wallet_info` — اطلاعات هر آدرس در Web3
بدون نیاز به هیچ کلیدی موجودی هر آدرسی رو توی اتریوم، BSC و پلی‌گان چک می‌کنه.

### 🔗 `connect_wallets` — 🎯 **ویژگی اصلی پروژه**
دو تا آدرس می‌دی بهش، و این کارها رو می‌کنه:
1. موجودی هر دو رو چک می‌کنه
2. تشخیص می‌ده قرارداد هوشمنده یا کیف پول معمولی
3. تراکنش‌های مشترک بینشون رو جستجو می‌کنه
4. خلاصه‌ای از ارتباطات رو نشون می‌ده

### 📊 `wallet_comparison` — مقایسه دو تا آدرس کنار هم
### 🪙 `token_balance` — موجودی توکن ERC-20 هر آدرس
### ℹ️ `transaction_info` — جزئیات کامل یه تراکنش
### 🔍 `is_contract` — تشخیص Contract یا EOA

## 🌐 بلاکچین‌های پشتیبانی شده

| بلاکچین | واحد | RPC عمومی |
|----------|------|-----------|
| **اتریوم** | ETH | `ethereum-rpc.publicnode.com` |
| **زنجیره هوشمند بایننس (BSC)** | BNB | `bsc-dataseed1.binance.org` |
| **پلی‌گان** | MATIC | `polygon-rpc.com` |

**همه رایگان** — بدون API Key، بدون ثبت‌نام.

---

## 🧪 یه مثال واقعی

برای اطلاعات یه آدرس:
```bash
echo '{"jsonrpc":"2.0","id":1,"method":"tools/call","params":{"name":"wallet_info","arguments":{"address":"0x1234567890abcdef1234567890abcdef12345678"}}}' | python3 thesis_wallet_connector.py 2>/dev/null
```

برای اتصال دو آدرس به هم:
```bash
echo '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"connect_wallets","arguments":{"wallet_a":"0x1234567890abcdef1234567890abcdef12345678","wallet_b":"0xaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa","chain":"bsc"}}}' | python3 thesis_wallet_connector.py 2>/dev/null
```

## 🧠 معماری فنی: چطور بدون API Key کار می‌کنه؟

**Web3 یعنی همینه — دسترسی مستقیم به بلاکچین بدون نیاز به واسط.**

```
آدرس شما → MCP Server (پایتون) → RPC عمومی بلاکچین → اطلاعات بلاکچین → خروجی
```
